## Installing Gurty

1. Start by installing rust. Install it at "https://www.rust-lang.org/tools/install"
2. Clone the Gurted Github repository.

##### git clone https://github.com/outpoot/gurted

3. Enter protocol/cli in the repository.
   
##### cd protocol/cli

4. Run cargo

##### cargo install --release

You should now able to run all gurty commands!
